# Ecoute Client

Survey Generation Application


## Setup

If you don't have Node.js, install it by following the instructions at `https://nodejs.org/`.

```sh
npm install -g purescript
npm install -g esbuild
npm install -g spago
```


## Build

```sh
spago build
spago bundle-app --to dist/js/index.js
```


## Lunch

```sh
docker compose up
```
