# Ecoute 

Survey Generation Application

## Setup

Access `https://www.haskell.org/ghcup/` and install GHCup.
Check ghcup list to see if ghc, hls, cabal is installed.

## Build

```sh
cabal build
```

## Launch

```sh
cabal run ecoute
```
